## Module Entity

### Description

This table describes the modules that are in courses

### Contents

* ID
    - **Required**
    - Automatically generated id
    - With prefix `app-`
    - Unique for each record
* Name
    - **Required**
    - `Single line text`
    - Title for the module
* Description
    - **Optional**
    - `Text Field`

### Rules

* Cannot have a duplcate `Name`
